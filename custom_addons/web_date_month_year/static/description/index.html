Odoo field date only month year

Odoo web widget, allow users only select month or year on Date field in UI.

Note: This is a widget option to hide date part, the value in the database/code still remain unchanged.

Usages

in tree and form view, add more "options" attribute into the field

options="{'datepicker': {'showType':'months'}}": only show month

options="{'datepicker': {'showType':'years'}}": only show year

That's it, the popup window of field date will only show month or year. In the tree view, the value is applied as well
